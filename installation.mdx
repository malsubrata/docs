## Automatic Installation
1. In your WordPress admin panel, go to **Plugins → Add New**.  
2. Search for **TeraWallet** or **Woo Wallet**.  
3. Click **Install Now**, then **Activate**.  

## Manual Installation
1. Download the plugin from [WordPress.org](https://wordpress.org/plugins/woo-wallet/).  
2. Upload the `.zip` via **Plugins → Add New → Upload Plugin**.  
3. Click **Install Now**, then **Activate**.  

## Setup
- A **Wallet** tab will appear under **WooCommerce → Settings**.  
- Configure wallet funding options, cashback rules, and notifications.  
- Users will see a **Wallet** tab in **My Account**.
