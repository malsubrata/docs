This page lists important **actions and filters** provided by TeraWallet.

| Hook | Type | Description |
|------|------|-------------|
| `woo_wallet_before_transaction` | Action | Fires before a wallet transaction is created. |
| `woo_wallet_after_transaction` | Action | Fires after a wallet transaction is recorded. |
| `woo_wallet_cashback_amount` | Filter | Modify cashback amount before crediting wallet. |
| `woo_wallet_partial_payment_enabled` | Filter | Enable/disable partial payments dynamically. |
