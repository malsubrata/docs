# TeraWallet (Wallet for WooCommerce)

TeraWallet is a WooCommerce wallet and credit system that allows customers to **add funds, make partial payments, transfer money, and earn cashback** directly within your store.

## ✨ Features
- Wallet payment gateway for WooCommerce checkout  
- Add funds via standard WooCommerce payment methods  
- Cashback rewards per product, category, or global rule  
- Refunds directly to wallet balance  
- Peer-to-peer wallet transfers  
- Partial payments (combine wallet + another payment method)  
- REST API support  
- Multisite & WPML support  

## ⚙️ Minimum Requirements
- PHP 7.4 or newer (8.0+ recommended)  
- WordPress 6.0 or newer  
- WooCommerce 7.2 or newer
